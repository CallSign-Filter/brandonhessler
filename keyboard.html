Universal Mouse-less Guide to Development

IntelliJ

  • Ctrl + Shift + N - Find File
  • Ctrl + 1 - Bring focus to Project window
  • ESC - Bring focus back to editor
  • Option (Alt) + F12 - Hide/ Show Terminal
  • Ctrl + Alt + L - Pretty format JSON
  • Ctrl + Alt + I - Pretty format Indents

Chrome

  • Ctrl + W - Close Tab
  • Ctrl + Shift + A - Open tab search
  • Tab + arrows + Tab + Enter - Close tab in tab search
  • Ctrl + Shift + [ || Ctrl + Shift + ] - Cycle through tabs
  • Ctrl + Alt + I - Open Dev Console

Others

  • Slack - Ctrl + K - Jump to conversation

  • ESC - Bring focus back to editor
