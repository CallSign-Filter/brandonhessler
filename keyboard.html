Universal Mouse-less Guide to Development

IntelliJ

  • ⌘ + ⇧ + N - Find File
  • ⌘ + 1 - Bring focus to Project window
  • ESC - Bring focus back to editor
  • ⌥ + F12 - Hide/ Show Terminal
  • ⌘ + ⌥ + L - Pretty format JSON
  • ⌘ + ⌥ + I - Pretty format Indents

Chrome

  • ⌘ + W - Close Tab
  • ⌘ + ⇧ + A - Open tab search
  • Tab + arrows + Tab + Enter - Close tab in tab search
  • ⌘ + ⇧ + [ || ⌘ + ⇧ + ] - Cycle through tabs
  • ⌘ + Alt + I - Open Dev Console

Others

  • Slack - ⌘ + K - Jump to conversation

Programs

  • Raycast
  • Better Touch Tool
  • Contexts
    • ⌥ + Tab - window switcher
    • ⌃ + Space - window searcher
